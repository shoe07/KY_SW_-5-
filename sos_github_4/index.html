⚡ AED 사용법

1
전원 켜기
뚜껑을 열거나 전원 버튼을 누릅니다. 음성 안내가 즉시 시작됩니다.
2
패드 부착
오른쪽 쇄골 아래 + 왼쪽 겨드랑이 아래. 젖은 피부는 닦고 부착합니다.
3
심장리듬 분석
"분석 중" 안내 시 모든 사람이 환자에게서 즉시 손을 뗍니다.
4
제세동 버튼
주변을 물리고 깜빡이는 버튼을 누릅니다. 충격이 전달됩니다.
5
CPR 재개
즉시 CPR 재개: 가슴 압박 30회 + 인공호흡 2회. 119 대원 도착 전까지 반복합니다.
